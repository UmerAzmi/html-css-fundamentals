# CSS Fundamentals

A collection of topic-focused CSS files covering core styling principles, layout techniques, UI components, and modern design behavior.

## Overview

This folder contains structured CSS learning files organized by concept. Each file focuses on a specific area of CSS and is intended to be studied alongside corresponding HTML demonstration pages.

## Topics Covered

* **1.css** — Colors, Fonts, Size Units, Borders, Shadows
* **2.css** — Margin, Float, Overflow, Display
* **3.css** — Positioning, Background
* **4.css** — Combinators, Pseudo-classes, Pseudo-elements
* **5.css** — Navigation Bar, Dropdown, Pagination
* **6.css** — Website Layout: Header, Footer, Navbar, Section, Aside, Article
* **7.css** — Flexbox
* **8.css** — Transitions

## How to Use

For effective learning:

1. Open the corresponding HTML files in a web browser to view the layout and styling.
2. Open the CSS files in a code editor to study the styles and inline comments explaining each concept.

No server or build tools are required.

## Notes

* Files are organized for conceptual progression from foundational styling to modern layout techniques.
* CSS in this folder supports demonstration pages and focuses on understanding styling behavior.
* This collection represents documented knowledge and project work. Additional skills and experience exist beyond what is included here.
