# Profile Card UI

A modern profile card component built using HTML and CSS, showcasing a clean layout, social links, action buttons, and user analytics in a compact card design.

## Features

* Modern card-based UI design
* Circular profile image with styled header section
* Social media icon links
* Action buttons (Resume and Message)
* Analytics section displaying experience highlights
* Hover animations and smooth transitions
* Fully responsive centered layout

## Technologies Used

* HTML5
* CSS3 (Flexbox, transitions, shadows, pseudo-elements)
* Boxicons for icons

## Purpose

This project was created as part of early HTML/CSS practice to learn component-based UI design, layout alignment, and interactive styling techniques.

## How to Run

1. Download or clone the project.
2. Open `index.html` in any modern web browser.
