# Projects

This directory contains a collection of small web projects built while learning and practicing HTML, CSS, and basic JavaScript. Each project focuses on specific UI components, layout techniques, responsiveness, or interactive behavior.

These projects demonstrate progressive improvement in front-end development skills and real-world design implementation.


## Included Projects

### Credit Card UI

An interactive credit card component with a 3D flip animation, glassmorphism styling, and realistic front and back details.

### Image Gallery

A multi-page wallpaper gallery showcasing layout design, styling, and pagination using pure HTML and CSS.

### Responsive Landing Page

A modern product-style landing page featuring responsive layout, mobile navigation, and a hero section with call-to-action.

### Profile Card

A compact profile card component displaying user information, social links, action buttons, and analytics indicators.

### Responsive Landing Page (Portfolio Theme)

A personal portfolio landing page with animated text, social links, and mobile navigation.


## Purpose

The goal of this folder is to apply learned concepts through hands-on projects, moving from simple layouts to more complete UI designs.


## How to Use

Each project is self-contained.

1. Navigate to a project folder.
2. Open the `index.html` file in a web browser.

No build tools or installation are required.


## Notes

* All projects are built using vanilla HTML, CSS, and minimal JavaScript.
* Assets and styles are stored within their respective project folders.
* These projects are part of documented learning progress and practical experimentation.
