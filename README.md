# HTML & CSS Fundamentals

A structured collection of learning materials, practice files, and small projects created while building a strong foundation in front-end web development using HTML, CSS, and basic JavaScript.

This repository focuses on understanding core concepts, layout techniques, responsiveness, and modern UI design patterns through hands-on practice.

## Repository Structure

### html/

Contains HTML learning files covering semantic structure, forms, elements, and page composition. These files include detailed comments explaining how different tags and features work.

### css/

Contains CSS learning notes and practice stylesheets organized by topics such as layout, positioning, selectors, Flexbox, and transitions.

### projects/

A collection of small web projects that apply the learned concepts in real UI designs, including responsive layouts, components, and interactive elements.

## Purpose

The goal of this repository is to document the learning process while developing practical front-end skills. It demonstrates progression from basic markup to fully styled and responsive interfaces.

## How to Use

* Explore the **html** and **css** folders to study individual concepts.
* Open any file directly in a web browser to see the result.
* Navigate to the **projects** folder to view complete mini-projects.

No build tools or installation are required.

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* Google Fonts
* Font Awesome / Icon Libraries

## Notes

* All files are self-contained and organized by topic.
* Projects are designed to run locally without additional dependencies.
* This repository represents documented learning progress and practical experimentation.
